# Public Share Package

This folder contains a sanitized public release of the evaluation dataset and summary.

Files:
- `evaluation_dataset_public.json` (full dataset)
- `evaluation_dataset_public.csv` (spreadsheet-friendly format)
- `evaluation_summary_public.json` (headline metrics)
- `dataset_card_public.md` (human-readable overview)
- `linkedin_post_template.txt` (ready-to-post draft)
